---
title: "Get All Users"
api: "GET /api/users"
description: "Retrieve a list of all users in the system"
---

Retrieve a list of all users registered in the Adoptme platform. This endpoint returns all user records with their associated information including pets.

## Response

<ResponseField name="status" type="string">
  Status of the response. Returns "success" on successful requests.
</ResponseField>

<ResponseField name="payload" type="array">
  Array of user objects.
  
  <ResponseField name="_id" type="string">
    Unique identifier for the user (MongoDB ObjectId).
  </ResponseField>
  
  <ResponseField name="first_name" type="string" required>
    User's first name.
  </ResponseField>
  
  <ResponseField name="last_name" type="string" required>
    User's last name.
  </ResponseField>
  
  <ResponseField name="email" type="string" required>
    User's email address (unique).
  </ResponseField>
  
  <ResponseField name="password" type="string" required>
    User's password (hashed).
  </ResponseField>
  
  <ResponseField name="role" type="string">
    User's role in the system. Defaults to "user".
  </ResponseField>
  
  <ResponseField name="pets" type="array">
    Array of pet references associated with the user. Each item contains a pet ID referencing the Pets collection.
  </ResponseField>
</ResponseField>
