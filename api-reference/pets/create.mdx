---
title: "Create Pet"
api: "POST /api/pets"
description: "Create a new pet without an image"
---

This endpoint creates a new pet in the adoption system with basic information.

### Body

<ParamField body="name" type="string" required>
  The name of the pet.
</ParamField>

<ParamField body="specie" type="string" required>
  The species of the pet (e.g., "dog", "cat", "bird", "rabbit").
</ParamField>

<ParamField body="birthDate" type="string" required>
  The birth date of the pet. Can be in ISO 8601 format or any valid date string.
</ParamField>

### Response

<ResponseField name="status" type="string">
  The status of the response. Returns `"success"` on successful creation.
</ResponseField>

<ResponseField name="payload" type="object">
  The created pet object.
  
  <ResponseField name="_id" type="string">
    The unique identifier for the newly created pet.
  </ResponseField>
  
  <ResponseField name="name" type="string">
    The name of the pet.
  </ResponseField>
  
  <ResponseField name="specie" type="string">
    The species of the pet.
  </ResponseField>
  
  <ResponseField name="birthDate" type="string">
    The birth date of the pet in ISO 8601 format.
  </ResponseField>
  
  <ResponseField name="adopted" type="boolean">
    Whether the pet has been adopted. Defaults to `false` for new pets.
  </ResponseField>
</ResponseField>

<RequestExample>
```bash cURL
curl --request POST \
  --url https://api.adoptme.com/api/pets \
  --header 'Content-Type: application/json' \
  --data '{
    "name": "Buddy",
    "specie": "dog",
    "birthDate": "2021-07-15"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.adoptme.com/api/pets', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    name: 'Buddy',
    specie: 'dog',
    birthDate: '2021-07-15'
  })
});

const data = await response.json();
```

```python Python
import requests

response = requests.post(
  'https://api.adoptme.com/api/pets',
  json={
    'name': 'Buddy',
    'specie': 'dog',
    'birthDate': '2021-07-15'
  }
)

data = response.json()
```
</RequestExample>
