  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get up and running with the Adoptme API in minutes
  </p>

  <Steps>
    <Step title="Clone the repository">
      Clone the project and install dependencies to get started locally.

      ```bash
      git clone https://github.com/GitRepoFabi/coderhouse-backendIII.git
      cd coderhouse-backendIII
      npm install
      ```
    </Step>

    <Step title="Configure environment variables">
      Create a `.env` file in the root directory with your MongoDB connection details.

      ```bash
      PORT=3000
      MONGO_URL=mongodb://localhost:27017
      DB_NAME=adoptme
      ```

      <Note>
        You need Node.js version 22 or higher to run this project. Check your version with `node --version`.
      </Note>
    </Step>

    <Step title="Start the server">
      Launch the development server and verify it's running.

      ```bash
      npm start
      ```

      The API will be available at `http://localhost:3000`. You can access the interactive API documentation at `http://localhost:3000/docs`.
    </Step>

    <Step title="Make your first request">
      Test the API by creating a new user.

      ```bash
      curl -X POST http://localhost:3000/api/sessions/register \
        -H "Content-Type: application/json" \
        -d '{
          "first_name": "John",
          "last_name": "Smith",
          "email": "john@example.com",
          "password": "password123"
        }'
      ```

      <Accordion title="Example response">
        ```json
        {
          "status": "success",
          "payload": {
            "_id": "507f1f77bcf86cd799439011",
            "first_name": "John",
            "last_name": "Smith",
            "email": "john@example.com",
            "role": "user",
            "pets": []
          }
        }
        ```
      </Accordion>
    </Step>
  </Steps>
